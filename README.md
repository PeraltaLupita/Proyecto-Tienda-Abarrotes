Clonar el Repositorio:
git clone https://github.com/ic8022569-source/Tienda-Abarrotes.git

Instalar node:
descargando node desde su pagina oficial(https://nodejs.org/es)
siguiendo la instalacion como lo marca la aplicacion

Acceder al Directorio e Instalar Dependencias:
cd Tienda-Abarrotes
npm install

Ejecutar el Servidor:
node server.js
